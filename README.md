# Rock Revival Watch Website

A luxury watch brand website built with **React** and **React-Bootstrap**, showcasing premium collections, categories. 

This project demonstrates modern UI/UX design, responsive layouts, and React routing.

##  Tech Stack

- **React**  
- **React-Bootstrap**  
- **React-Router-Dom**  
- **JavaScript (ES6+)**  
- **CSS3 / Flexbox / Grid**
